## 👀프로젝트 소개

사용자들의 건강한 라이프스타일 유지를 목표로 한 데일리 운동 인증 웹사이트  
오운완 인증, 프로틴 추천, 루틴 공유, 헬스장 리뷰 4가지 채널을 제공하며
게시글 작성을 통해 건강과 운동 정보를 공유하고, 좋아요, 댓글, 팔로우 기능으로 사용자 간 활발한 커뮤니티를 형성합니다.
BMI 계산기, 운동 인증 캘린더 및 원 그래프 기능을 통해 사용자들에게 운동 동기를 부여합니다.

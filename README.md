건강한 라이프스타일을 위한 데일리 운동 인증 커뮤니티

- **4가지 주요 채널 제공** : 오운완 인증, 프로틴 추천, 식단 공유, 헬스장 리뷰
  - 게시글 작성으로 건강 및 운동 정보를 공유
  - 좋아요, 댓글, 팔로우 기능으로 **활발한 커뮤니티 형성**
- **운동 동기 부여 기능**:
  - BMI 계산기
  - 운동 인증 캘린더 및 원 그래프
